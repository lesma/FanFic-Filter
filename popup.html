Summary Filters
[Show Instructions]

Use [ Add Filter ] to add filter criterion.

Type in a particular criterion in the first box and a number to check against in the last box.

A criterion can be of the form <var> or <var>/<var>.

<var> can be any of the below.

 chapsChapters
 wordsWords
 revsReviews
 favsFavs
 follsFollows

Use the Tag Box to add summary filter tags.

Type in a tag to filter the summary and press Enter.

Use / for pairings. For example, Naru/Sasu will check for

  •  NaruSasu 
  •  Naru Sasu 
  •  Naru/Sasu 
  •  NaruXSasu 
  •  Naru!Sasu 
  •  Naru-Sasu 
  •  SasuNaru 
  •  Sasu Naru 
  •  Sasu/Naru 
  •  SasuXNaru 
  •  Sasu!Naru 
  •  Sasu-Naru 

Filter tags can be  positive  or  negative .

 Negative  filter tags will be excluded from filtered stories.

 Positive  filter tags will be always included in filtered stories, and stories not containing them will be filtered out.

Use Settings to change add-on configuration.

  disable to disable/enable filter functions.

  show story images to show/hide images next to stories. Showing images might make things a bit slower!

  default tags to select the default filter tag.

     If negative is default, use + before tag for  positive  tag. Example, +positive for  positive .

     If positive is default, use - before tag for  negative  tag. Example, -negative for  negative .
